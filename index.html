ВСТРОЕННАЯ ЛОГИКА РЕЖИМОВ

📘 STUDY MODE (Режим Обучения)
  • Аудио проигрывается 2 раза свободно.
  • Блокировка перемотки срабатывает только на 3-й попытке.
  • Доступна кнопка Slow Motion (0.8x) без искажения высоты голоса (pitch-preserving).
  • Таймер: Без ограничений.
  • Экран: Не блокируется по времени.
🎓 MASTER MODE (Режим Мастера)
  • Включена функция Loop (цикличное воспроизведение аудио).
  • Полный контроль над прогресс-баром и перемоткой.
⏱️ TEST MODE (Режим Экзамена)
  • Перемотка аудио полностью заблокирована.
  • Жесткий таймер: 2 минуты (120 сек).
  • По истечении времени: Аудио ставится на паузу.
  • Экран блокируется сообщением: "Time is up".
